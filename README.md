Sistema Web para Clínicas

Este projeto consiste em um site desenvolvido para clínicas divulgarem informações importantes para pacientes. A plataforma permite que a clínica apresente seus serviços, planos de saúde aceitos e dados de contato de forma organizada e acessível.

O sistema possui funcionalidades como:

📍 Endereço da clínica e informações de localização
🏥 Lista de serviços oferecidos pela clínica
💳 Planos de saúde aceitos
📞 Informações de contato para atendimento
🖥️ Interface simples e fácil de usar para pacientes

O objetivo do projeto é facilitar o acesso às informações da clínica, permitindo que pacientes encontrem rapidamente os serviços disponíveis e os convênios aceitos.
